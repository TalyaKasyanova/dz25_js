Асинхронне підключення - якщо потрібно завантажувати JavaScript код асинхронно, щоб не затримувати завантаження сторінки, можна використовувати атрибут async Відкладене підключення - якщо потрібно завантажити JavaScript код після завантаження сторінки, можна використовувати атрибут defer Інлайновий спосіб підключення - код JavaScript можна додавати в атрибут HTML елементів, таких як onclick, onload, тощо
